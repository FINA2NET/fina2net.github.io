TIMS@Tajikistan [ July 2007 - March 2009 ]

  • Project Name: FinA @ TIMS@Tajikistan(Public Finance Management Capacity Building)
  • Country studied: Tajikistan
  • Sponsor: European Commission
  • Partners: Sofreco, Case, SRC Period: 2017-2019
  • Language: Russian/English
  • IT Environment: OS - Windows, Database - MS SQL Server 2005
  • Description: This project is part of the TA effort to improve Public Finance Management, in particular dealing with Public Internal Financial Control and Treasury Management Information System. It will build on the work started by earlier EC projects and develop the legislation and institutional capacity for PIFC in the Ministry of Finance. It will help in the development of functioning Internal Audit Units in four line ministries. This Project will audit the MIS in place and assist the Tajik Government in elaborating an MIS Development Plan.
